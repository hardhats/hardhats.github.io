$$IEN2CODE^XUA4A72(): Get VA Code

Reference Type: Supported, Category: User, ICR#: 1625

Description

The $$IEN2CODE^XUA4A72 extrinsic function returns the VA CODE from the PERSON CLASS (#8932.1) file that corresponds to the Internal Entry Number (IEN) passed in. If the IEN passed in does not match a valid entry in the PERSON CLASS (#8932.1) file, an empty string is returned.

NOTE: This API was released with Kernel Patch XU*8.0*27.

Format

  $$IEN2CODE^XUA4A72(ien)

Input Parameters

ien:

(required) Internal Entry Number (IEN) in the PERSON CLASS (#8932.1) file.

Output

returns:

Returns the VA CODE.
