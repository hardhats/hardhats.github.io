$$IDX^XUAF4(): Institution IEN (Using Coding System & ID)

Reference Type: Supported, Category: Institution File, ICR#: 2171

Description

The $$IDX^XUAF4 extrinsic function returns the Internal Entry Number (IEN) of an INSTITUTION (#4) file entry for a given coding system and identifier (ID) pair.

Format

  $$IDX^XUAF4(cdsys,id)

Input Parameters

cdsys:

(required) CDSYS is an existing coding system of the INSTITUTION (#4) file. To see the existing coding system in the file:

  >D CDSYS^XUAF4(.Y)
id:

(required) ID is the identifier associated with the coding system. The station number, for example, is the identifier for the VASTANUM coding system and NPI number is the ID for the NPI coding system.

Output

returns:

Returns the INSTITUTION (#4) file Internal Entry Number (IEN) associated with the given coding system and identifier (ID).
