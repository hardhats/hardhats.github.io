VistA has been growing steadily since the mid-80s and today represents an unparalleled number of integrated HIS applications. Designed from the beginning to focus on clinical aspects of healthcare, the applications share a common set of files such as the patient and provider files. Integration, from the very beginning of its design, permits access to clinical data without duplication, thereby increasing accuracy and reducing maintenance costs.

Work is presently under way to provide further enhancements to the computerized patient record system (CPRS) within VistA that can be accessed from a physician's workstation.  CPRS currently provides a Delphi generated Graphical User Interface (GUI) that uses the RPC Broker to communicate with the M server.

A regularly updated collection of monographs that describe the VistA applications can be found here.

Platinum Account: Complete VistA Installation

The VistA applications that are available under the Freedom of Information Act, come preinstalled and patched in a set of M routines and globals that can be downloaded from the VA's web site. The media is released with the latest patches, as KIDS distributions in folders, as well as incorporated in the installed VistA applications. The VA's software web site contains the most up to date list of patches, software releases, etc. The documentation is also available via a very well organized Library page.

Here are sources where you can get assistance in getting Platinum installed and operational on your computer.

  • The Hardhats Installing VistA documentation.
    (Usually the most current and detailed source.)

  • The readme file for the Platinum pre-installed applications describes the files that are and are not included.

  • Detailed instructions, at another website, on how to get the Platinum installation up and running.
