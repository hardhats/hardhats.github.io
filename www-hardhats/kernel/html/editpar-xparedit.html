EDITPAR^XPAREDIT(): Edit Single Parameter

Reference Type: Supported, Category: Toolkit—Parameter Tools, ICR#: 2336

Description

The EDITPAR^XPAREDIT API edits a single parameter.

Format

  EDITPAR^XPAREDIT(parameter)

Input Parameters

parameter:

(required) Can be passed in external or internal format. Identifies the name or internal entry number (IEN) of the parameter as defined in the PARAMETER DEFINITION (#8989.51) file.

Output

returns:

Returns requested parameter.
